Objetivo

Desenvolver um sistema automatizado de irrigação que monitore e controle a umidade em múltiplos pontos de uma plantação, garantindo uma irrigação eficiente em áreas maiores e reduzindo o desperdício de água.

Funcionamento

O sistema utiliza um sensor de umidade conectado ao Arduino. Quando o solo está seco, uma bomba é acionada para liberar água.

Componentes

  • Arduino Mega ou ESP32: Mais pinos para múltiplos sensores
  • Sensores de Umidade do Solo: Colocados em diferentes partes do terreno
  • Módulo(s) Relé 4 canais: Controla várias bombas ou válvulas
  • Bombas d’água maiores ou válvulas solenoides: Para dividir a irrigação em zonas
  • Fonte de alimentação externa: Fonte robusta para alimentar bombas e Arduino
  • Reservatório ou caixa d’água grande: Armazena grande volume de água
  • Mangueiras e conectores: Distribui água entre as zonas
  • Cabos longos ou módulos sem fio (opcional): Para comunicação entre sensores distantes
  • Display LCD/OLED (opcional): Mostra dados no campo
  • Módulo RTC (relógio de tempo real) (opcional): Para agendamento da irrigação

Exemplo de Código Arduino

int sensorUmidade = A0;
int bomba = 8;

void setup() {
    pinMode(bomba, OUTPUT);
    Serial.begin(9600);
}

void loop() {
    int umidade = analogRead(sensorUmidade);
    Serial.println(umidade);
    
    if (umidade < 400) {
        digitalWrite(bomba, HIGH); // Liga a bomba
    } else {
        digitalWrite(bomba, LOW); // Desliga a bomba
    }

    delay(2000);
}
