Cálculo do Consumo de Água em JavaScript


O código abaixo simula a leitura dos dados de irrigação e calcula o total de água utilizada na semana e uma estimativa mensal, considerando o tempo de irrigação diário e o consumo da bomba. 


// Consumo da bomba por minuto (em litros)
const litrosPorMinuto = 1.5;

// Minutos de irrigação por dia (7 dias)
const minutosPorDia = [10, 8, 12, 5, 7, 9, 11];

// Cálculo do total semanal
let totalSemanal = 0;
for (let minutos of minutosPorDia) {
    totalSemanal += minutos * litrosPorMinuto;
}

// Estimativa mensal (4 semanas)
const totalMensal = totalSemanal * 4;

// Exibir resultados no console
console.log("Total de água usada na semana: " + totalSemanal + " litros");
console.log("Estimativa de água usada no mês: " + totalMensal + " litros");
Como funciona o cálculo?
  • litrosPorMinuto: Quantidade de água (em litros) que a bomba consome por minuto.
  • minutosPorDia: Array com o tempo de irrigação (em minutos) para cada dia da semana.
  • totalSemanal: Soma do consumo diário multiplicado pelo consumo da bomba.
  • totalMensal: Estimativa baseada em 4 semanas.
